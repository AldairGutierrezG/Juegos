# 📚 Aprender Jugando

## 💖 Dedicado a Sahily Melisa Gutierrez Chairez

Este pequeño proyecto fue creado con mucho cariño para **Sahily Melisa Gutierrez Chairez**, una niña maravillosa de **4 años**, con el objetivo de ayudarla a aprender mientras juega, explora y se divierte.

La aplicación busca que el aprendizaje de **letras, números y figuras geométricas** sea una experiencia sencilla, visual y agradable, permitiendo que los niños desarrollen su curiosidad natural a través de la interacción.

Cada línea de código de esta aplicación fue pensada como una herramienta para acompañar sus primeros pasos en el mundo del conocimiento.

---

# 🎯 Objetivo del proyecto

Crear una aplicación educativa simple que permita a los niños:

* Reconocer **letras**
* Aprender **números hasta el 100**
* Identificar **figuras geométricas**
* Desarrollar **memoria visual**
* Practicar el **trazo de letras y números**

Todo mediante una interfaz amigable, clara y pensada especialmente para niños de **4 a 7 años**.

---

# 🧩 Funcionalidades

La aplicación incluye los siguientes módulos:

## 📚 Letras

* Visualización de letras grandes
* Palabra asociada (ejemplo: **A de Árbol**)
* Reproducción de sonido
* Área para dibujar la letra

## 🔢 Números

* Números desde **0 hasta 100**
* Pronunciación del número
* Área para practicar el trazo

## 🔺 Figuras geométricas

Reconocimiento de:

* Cuadrado
* Triángulo
* Círculo
* Estrella

Con espacio para dibujar cada figura.

## 🧠 Juego de memoria

Un pequeño juego donde el niño debe encontrar pares entre:

* Letras
* Números
* Figuras

Esto ayuda a desarrollar:

* Memoria visual
* Atención
* Reconocimiento de patrones

---

# ✏ Área de dibujo

La aplicación incluye un **canvas interactivo** donde el niño puede:

* Dibujar con el dedo o el mouse
* Copiar la letra o número mostrado
* Borrar y volver a intentar

Esto ayuda a desarrollar:

* Coordinación mano-ojo
* Motricidad fina
* Reconocimiento visual

---

# ⭐ Refuerzo positivo

Cuando el niño completa una actividad aparece una **estrella de felicitación**, reforzando la motivación y el aprendizaje positivo.

# 🛠 Tecnologías utilizadas

Este proyecto fue desarrollado utilizando tecnologías web simples y accesibles:

* **HTML5**
* **CSS3**
* **JavaScript**
* **Canvas API**
* **Speech Synthesis API**

Esto permite que la aplicación funcione en:

* Computadoras
* Tablets
* Teléfonos móviles

---

# 📱 Conversión a aplicación Android

La aplicación puede convertirse en APK utilizando herramientas como:

* PWABuilder
* Capacitor
* Android Studio

Esto permite instalarla directamente en un teléfono Android.
